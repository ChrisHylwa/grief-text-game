A work of interactive fiction.

Room Wodplay or Surrealist project was created with Inform and has IFID 676938C5-CE16-4BA7-B913-B3D51F945CF1. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
